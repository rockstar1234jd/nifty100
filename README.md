# Nifty 100 Financial Intelligence Platform

Sprint 1 data foundation scaffold for the Nifty 100 project.

## What is included

- A clean `src/` package layout for ETL, analytics, dashboard, and API work
- Starter ETL modules for normalisation, loading, and validation
- A project-local pytest configuration and focused unit tests
- The directory structure expected by the Sprint 1 brief

## Run the foundation checks

```bash
pytest tests/etl tests/dq -v
```
